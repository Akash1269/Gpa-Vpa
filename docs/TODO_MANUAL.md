# TODO — Manual Actions Required

Tasks that need your personal action (can't be automated).

## Before First Build

- [ ] Create Expo account at https://expo.dev/signup
- [ ] Install EAS CLI: `npm install -g eas-cli`
- [ ] Run `eas login` to authenticate
- [ ] Run `eas init` to get real project ID (replaces placeholder in app.json)
- [ ] Design and create app icon (1024x1024px, PNG, no alpha, no rounded corners)
- [ ] Create splash screen image (use same icon or a branded version)

## Before Store Submission

- [ ] Host Privacy Policy at a public URL (options: GitHub Pages, a simple website, or Notion public page)
- [ ] Fill in Google Play Data Safety form (answer: "No data collected or shared")
- [ ] Fill in Apple App Privacy section (answer: "Data Not Collected")
- [ ] Take app screenshots on multiple device sizes:
  - iPhone 6.7" (iPhone 15 Pro Max)
  - iPhone 6.5" (iPhone 14 Plus)
  - iPhone 5.5" (iPhone 8 Plus)
  - iPad (if supporting tablets)
  - Android phone
- [ ] Write app store description (short + full)
- [ ] Choose keywords/categories for discoverability
- [ ] Complete age rating questionnaire (both stores)

## Accounts & Fees

- [ ] Google Play Developer account — $25 one-time (https://play.google.com/console)
- [ ] Apple Developer account — $99/year (https://developer.apple.com)

## After First Release

- [ ] Set up a support email for app store listings
- [ ] Monitor crash reports and reviews
- [ ] Plan update cadence (bug fixes, features from Priority 2-4)
